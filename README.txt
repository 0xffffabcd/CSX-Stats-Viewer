CSX Stats Viewer
================

CSX Stats Viewer is an editor for csstats.dat generated by the CSX module 
in AMX Mod X (http://www.amxmodx.org/)

Features
--------

* View/Edit the stats out of the game
* Search for players
* Delete multiple entries at once
* Merging players
